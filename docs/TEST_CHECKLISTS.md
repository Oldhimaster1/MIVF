# Test Checklists

## Player Runtime Smoke Test
1. Launch app.
2. Browser opens.
3. Open MIVF file.
4. Audio starts.
5. Pause/resume works.
6. Seek works.
7. Open settings and return.
8. Subtitles work when sidecar exists.
9. MoFlex path works when available.
10. Return to browser.

## UI Slice Test
1. Verify changed UI area visually.
2. Verify unchanged UI areas still render.
3. Verify no input behavior changes.
4. Verify alerts still render.
5. Verify playback and browser transitions remain stable.
