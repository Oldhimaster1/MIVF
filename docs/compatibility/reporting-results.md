# Reporting Results

*Placeholder — the formal reporting process isn't finalized yet.*

If you test MIVF on physical hardware, the following makes a result usable for
[Hardware Matrix](hardware-matrix.md):

- Exact console model (e.g. "New Nintendo 3DS," not just "3DS")
- Exact build ID or artifact SHA-256 you tested
- What you tested (which stages/features) and the result of each
- Whether the run completed or where it stopped
- Any visual glitches, slowdown, or crashes observed
- The preserved log file, if available

Open an issue on the GitHub repository with this information until a dedicated
submission process exists here.
