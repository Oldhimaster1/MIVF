# The Showcase

The Showcase is a deterministic, autorun demonstration built into Showcase-enabled
builds: it walks the Library, the DVD-style menu, Scene Selection, playback, the
Dashboard, real Theme/Color-Vision personalization, Resume, the idle screensaver, and a
closing title card — using the same real screens and real input handling a person would,
not a scripted fake. It exists specifically to give an honest, watchable demonstration of
what's actually implemented, gated behind a compile-time flag so it never ships in a
normal build.

**Media:** no public video capture exists yet. This page will hold the official
recording once one is produced from a lawful demonstration project and an identified
release candidate — see [Project Status](status.md) for what's been run so far
(emulator-confirmed, plus one preliminary physical-hardware run reported).

Cancel a running Showcase at any time by holding **B + START + SELECT**.
