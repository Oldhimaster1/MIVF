# Subtitles

**Status:** Build-verified, part of the frozen playback foundation (hardware-verified as
of the last tagged release, unchanged since — see [Validation Method](../compatibility/validation-method.md)).

- Place a `.srt` file next to the video: `yourvideo.srt`, plus `yourvideo.1.srt`,
  `yourvideo.2.srt`, … for additional tracks.
- Cycle tracks during playback with **Y** (see [Playback](playback.md)).
- Adjustable delay and on-screen position (Settings menu).
- Multi-line subtitle display is supported.

See [Files & Sidecars](../authoring/files-and-sidecars.md) for the exact sidecar naming
rules, and [Creating Subtitles](../authoring/subtitles.md) for authoring guidance.
