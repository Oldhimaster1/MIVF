MIVF Player is released under the **MIT License**. See [LICENSE](https://github.com/Oldhimaster1/MIVF/blob/main/LICENSE)
in the repository root for the full, authoritative text.

MIT's actual requirement is narrow: **the copyright notice and license text must be
preserved** in copies or substantial portions of the software. It does not, by itself,
require attribution/credit lines, notification, or any other condition.

## A distinction worth being precise about

[NOTICE.md](https://github.com/Oldhimaster1/MIVF/blob/main/NOTICE.md) additionally
*requests* reasonable attribution (a suggested credit line) when substantial portions of
this project are reused elsewhere. That request is real and appreciated, but it is
**not** the same thing as a mandatory MIT license condition — the only legally mandatory
requirement is preserving the copyright notice and license text per LICENSE itself.
`NOTICE.md`'s current wording ("you must ... provide reasonable credit") reads as
stricter than that, and is flagged here as needing a legal-wording pass so mandatory
requirements (from LICENSE) and requested courtesy (from NOTICE.md) are clearly
distinguished, rather than implying attribution is itself a license condition.
