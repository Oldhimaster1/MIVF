# Controls Reference

## File Browser

| Button | Action |
| :--- | :--- |
| D-Pad ↑/↓ | Move selection up/down |
| A | Open selected file |
| Y | Toggle favorite on selected file |
| B | Go back / exit browser |
| START | Exit app |

Scrolling through files: preview thumbnails are debounced — the preview loads after you stop on a file for ~200 ms. This keeps scrolling smooth on Old 3DS hardware.

## Playback

| Button | Action |
| :--- | :--- |
| A | Play / pause |
| ← / → | Seek backward / forward (~5 seconds) |
| Touch + drag on timeline | Scrub to any position |
| X | Cycle playback speed (0.5×, 0.75×, 1.0×, 1.25×, 1.5×, 2.0×) |
| B | A/B scene loop: press once to set A, twice to set B (loops A→B), third time to clear |
| Y | Cycle subtitle track (off → track 1 → track 2 → …) |
| L + D-Pad ↑/↓ | Increase / decrease audio volume |
| R + D-Pad ↑/↓ | Increase / decrease screen brightness |
| R + D-Pad ←/→ | Jump to previous / next chapter |
| SELECT | Open settings overlay |
| START | Stop playback and return to browser |

## Settings Menu

Open with SELECT during playback.

| Button | Action |
| :--- | :--- |
| D-Pad ↑/↓ | Move between settings |
| A / D-Pad ←/→ | Adjust selected setting |
| B / SELECT | Close settings and save |

Settings are saved when you close the menu, not on every value change. This prevents SD-card stalls during rapid adjustment.

### Settings Items

- Resume bookmarks (on/off)
- Playback speed
- Auto-advance (play next file in folder)
- Sleep timer (off / 15 / 30 / 45 / 60 / 90 / 120 min)
- Aspect ratio (FIT / STRETCH / NATIVE)
- Screen brightness (1–5)
- Auto dim (on/off)
- Theme (4 color themes)
- Font scale
- Force stereo 3D
- Subtitles (on/off)
- Subtitle track
- Subtitle delay (±ms)
- Subtitle position (low / middle / high)
- Chapters display (on/off)
- Favorites (on/off)
- Lid sleep allowed (on/off)
- Dim timeout
- Dim brightness level
- Debug overlay (on/off)

## Touch Controls

During playback, touch the bottom screen timeline area to scrub. The player shows a time bubble while dragging.
