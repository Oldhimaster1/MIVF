# Screenshots

Drop real screenshots here with these filenames and the main [README.md](../README.md) will pick them up automatically:

| Filename | Shows |
| :--- | :--- |
| `browser.png` | The file browser (thumbnails, favorites) |
| `playback_subtitles.png` | Playback with subtitles on |
| `settings.png` | The Settings menu |
| `controls_help.png` | The in-app controls help screen |

Capturing via Azahar (emulator) is the easiest route — no SD card shuffling needed. Real hardware captures work too (L+R, or whatever your CFW binds to screenshot).

PNG or JPG both work; the root `.gitignore` has an explicit exception for images in this folder so they won't get silently excluded.
